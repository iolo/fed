About fed

fed - Bitmap Font Editor

A simple, minimalistic font editor for retro computing.


Features:

  • 📂 Load bitmap fonts (.fnt, .bin)
  • ✏️ Pixel-perfect editing
  • ⌨️ Keyboard navigation
  • 💾 Save binary fonts
  • 📤 Export PNG font sheets
  • 🔧 Configurable font dimensions

Keyboard shortcuts:

  • Arrow keys: Navigate glyphs/pixels
  • Enter/Esc: Start/Finish glyph editing
  • Space: Toggle pixel
  • +/-: Zoom in/out (browser/editor based on mode)
  • Ctrl++/Ctrl+-: Zoom both panels
  • 0: Reset zoom to 100% (current panel)
  • Ctrl+0: Reset both panels zoom
  • F: Auto-fit (current panel)
  • Ctrl+F: Auto-fit both panels
